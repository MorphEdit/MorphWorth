# MorphWorth

A lightweight Minecraft plugin that automatically displays an item's worth inside its lore.  
Supports **Paper / Spigot / Purpur 1.21.4+**.

- Shows item worth directly in lore

- Customizable:
  - Lore format
  - Prefix
  - Decimal rounding
  - Update interval
- No dependencies
